# **App Name**: HealthGuardian AI

## Core Features:

- Vitals Monitoring: Continuously track heartbeat, pulse rate, and blood pressure via sensor data.
- Movement and Fall Detection: Analyze movement sensor readings to detect falls and unusual activity patterns.
- AI-Powered Medication Alerts: Use AI to remind patients to take their medication and notify relatives if doses are missed.
- Emergency Contact Notification: Automatically notify relatives if a fall or other emergency is detected.
- Secure Data Storage: Store health data securely using MongoDB.
- API Integration: Integrate with external health APIs to retrieve and display additional health information.
- Real-Time Data Visualization: Visualize health data in real-time, providing patients and caregivers with up-to-date information.

## Style Guidelines:

- Primary color: Light, calm blue (#A0D2EB) to evoke trust and serenity.
- Background color: Soft, desaturated blue (#E6F0F5) for a clean and uncluttered feel.
- Accent color: Warm coral (#FF7F50) to highlight important alerts and CTAs.
- Body font: 'PT Sans', a modern, humanist sans-serif for readability and a touch of warmth.
- Headline font: 'Space Grotesk', a clean and techy sans-serif, used sparingly.
- Use clear, simple icons to represent health metrics and alerts.
- Prioritize a clean, intuitive layout that is easy to navigate, with key data points immediately accessible.
- Subtle animations for data updates and notifications to provide feedback without being distracting.
